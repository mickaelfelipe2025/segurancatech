    IPv6 é a nova versão do protocolo de endereçamento da internet e foi criado para substituir o IPv4, que já está ficando sem endereços disponíveis.

    A internet funciona com base em endereços IP. Cada dispositivo conectado precisa de um identificador único para se comunicar com outros dispositivos na rede.

    Durante décadas, o protocolo utilizado foi o IPv4. Porém, com o crescimento explosivo da internet — smartphones, smart TVs, câmeras, sensores IoT, computadores e servidores — o número de endereços disponíveis começou a se esgotar.

    1) O problema do IPv4

    O IPv4 utiliza 32 bits para gerar endereços IP.

    Isso permite aproximadamente 4,3 bilhões de combinações possíveis.

    Na década de 90 isso parecia suficiente. Hoje não é.

    2) O que mudou com o IPv6?

    O IPv6 utiliza 128 bits para endereçamento.

    Isso permite um número praticamente ilimitado de endereços IP.

    3) Diferença visual entre IPv4 e IPv6

    • IPv4: 192.168.0.1
    • IPv6: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

    4) Cenário real

    Hoje, provedores utilizam NAT e CGNAT para compartilhar IPs IPv4 entre clientes.
