Golpe do QR Code Falso: como funciona e como evitar

Leitura: 8–11 min • Atualizado: 18/02/2026
Escanear um QR Code pode parecer inofensivo, mas hoje é uma das formas mais rápidas de cair em golpes digitais.

O golpe do QR Code falso está se tornando cada vez mais comum no Brasil. Ele funciona de forma simples: o criminoso cria um código que direciona a vítima para um site falso, página de pagamento ou até instala malware no dispositivo.

Como QR Codes não mostram o link real antes de serem escaneados, eles se tornaram uma ferramenta perfeita para criminosos aplicarem golpes de forma silenciosa e eficiente.

Como o golpe funciona

O golpista cria um QR Code malicioso e o coloca em locais estratégicos como:

  • Mesas de restaurantes
  • Postes
  • Estacionamentos
  • Cartazes
  • Comprovantes falsos

Ao escanear o código, a vítima pode ser redirecionada para:

  • Página de login falsa
  • Pagamento via PIX
  • Download de aplicativo malicioso
  • Site que coleta dados pessoais
QR Code não é sinônimo de segurança. Ele apenas esconde o link real.

Sinais de QR Code malicioso

  • QR Code colado por cima de outro
  • QR Code em locais improvisados
  • Link estranho após escaneamento
  • Página solicitando login ou pagamento

Como se proteger

  • Evite escanear QR Codes desconhecidos
  • Use apps que mostram o link antes de abrir
  • Confira se o código não foi adulterado
  • Evite fazer pagamentos após escanear
  • Sempre digite o site manualmente

Criminosos sabem que quanto mais rápido você agir, maior a chance de cair no golpe. Por isso, sempre pare e analise antes de escanear qualquer código.
