Sequestro de DNS (DNS Hijacking): como criminosos te redirecionam para sites falsos

Leitura: 22–27 min • Atualizado: 18/02/2026

Mesmo digitando corretamente o endereço do seu banco, você pode estar acessando um site falso sem perceber.

O DNS Hijacking, também conhecido como sequestro de DNS, é um ataque onde criminosos alteram a forma como seu dispositivo interpreta endereços de sites na internet. O DNS funciona como uma lista telefônica da internet, convertendo nomes de domínio como "banco.com" em endereços IP reais.

Quando esse sistema é comprometido, você pode ser redirecionado para um site falso mesmo digitando o endereço correto no navegador.

Como o ataque funciona tecnicamente

O criminoso pode alterar o DNS diretamente no roteador da vítima ou infectar o computador com malware capaz de modificar as configurações de rede.

Após isso, qualquer tentativa de acesso a determinados sites será redirecionada para páginas clonadas.

Cenário real

Você tenta acessar o site do seu banco digitando o endereço manualmente.

Porém, devido à alteração no DNS, você é levado para uma página falsa idêntica à original.

Ao inserir login e senha, essas informações são capturadas.

Uso dos dados roubados

  • Transferências via PIX
  • Compras online
  • Solicitação de empréstimos
  • Roubo de identidade

Consequências

A vítima pode ter contas bancárias esvaziadas em questão de horas.

Mesmo verificando o cadeado HTTPS, você pode estar em um site falso.

Primeiras 24h

  • Resetar roteador
  • Alterar DNS manualmente
  • Trocar senhas
  • Verificar antivírus

Plano de 7 dias

  • Atualizar firmware do roteador
  • Monitorar contas bancárias
  • Ativar alertas

Como evitar

  • Trocar senha do roteador
  • Atualizar firmware
  • Desativar acesso remoto
  • Usar DNS seguro

Checklist

  • Senha forte no roteador
  • Firmware atualizado
  • DNS confiável
  • Firewall ativo
